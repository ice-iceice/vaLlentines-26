# Thumbnail Images Folder

**This folder contains the SMALL BUTTON/ICON images that appear on your desktop.**

## Files to place here:

- `p1.jpg` (or .png, .jpeg, .webp, .gif)
- `p3.jpg`
- `p4.jpg`
- `p5.jpg`
- `p6.jpg`
- `p7.jpg`
- `letter.jpg` (exception file)
- `location.jpg` (exception file)

These are the **clickable thumbnail images** that users see on the desktop. They should be small, preview-sized images (typically 80x80px or similar).
